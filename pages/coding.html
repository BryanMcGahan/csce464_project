Coding

Haskell

In this online class, you'll learn the fundamentals of Haskell, a purely functional programming language known for its concise syntax and powerful type system. You'll start with the basics, including data types, functions, and control structures, and gradually work your way up to more advanced topics like higher-order functions, monads, and type classes

Javascript

This online class about JavaScript will teach you how to use the programming language to create dynamic and interactive websites. You will learn the fundamentals of JavaScript, including variables, functions, and control flow, and how to use them to add interactivity and functionality to your web pages.

CSharp

This online class about C# will introduce you to the fundamentals of programming with C#, covering topics such as syntax, data types, control structures, object-oriented programming, and more. By the end of the course, you will have a solid understanding of C# and be able to write basic programs using this popular programming language.

Frontend

This online class about frontend development will teach you how to design and create the user interface of websites using HTML, CSS, and JavaScript, and provide you with the skills needed to make your web pages interactive and visually appealing. You'll learn how to structure your code, optimize for performance, and troubleshoot common issues in order to create seamless and engaging user experiences.

Rust

This online class about Rust will introduce you to the Rust programming language and teach you how to write efficient, reliable, and safe code that can be used for systems programming, network programming, web development, and more. Through a combination of lectures, coding exercises, and projects, you will gain a solid understanding of Rust's syntax, data types, control flow, error handling, concurrency, and performance characteristics.

PHP

This online class about PHP will teach you how to use this popular scripting language to create dynamic web pages and web applications. Through interactive lessons and practical exercises, you'll learn about PHP syntax, functions, data types, control structures, database integration, and more.

Python

This online class will teach you the fundamentals of Python programming, including variables, loops, functions, and data structures. You'll also learn how to write and run Python code using popular development tools like Jupyter Notebook and Visual Studio Code.

C

This online class will teach you the fundamentals of the C programming language, including syntax, data types, control structures, functions, and pointers. By the end of the course, you will be able to write simple C programs and have a solid foundation for further study in this powerful language.

Cobol

This online class about COBOL is designed to teach students the fundamentals of the COBOL programming language, including syntax and program structure, with practical examples and hands-on assignments. By the end of the course, students should be able to write and execute basic COBOL programs.
